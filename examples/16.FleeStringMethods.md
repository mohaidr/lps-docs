# String Methods (Flee Expressions) Complete Examples

This document contains complete runnable plans for Flee string expressions.

In Flee, skipIf is evaluated as a C# expression, so string comparisons and string-method calls must operate on quoted string values.

---

## Flee with QJson (recommended)

QJson emits string placeholders as quoted values, which works naturally with Flee string methods.

```yaml
name: TestPlan
rounds:
  - name: FeatureRound
    numberOfClients: 1
    arrivalDelay: 1000
    runInParallel: false
    iterations:
      - name: usersQ
        httpRequest:
          capture:
            to: usersQ
            as: QJson
            makeGlobal: true
          url: https://jsonplaceholder.typicode.com/users
          method: GET
      - name: fleeUpperCheck
        httpRequest:
          skipIf: '$usersQ.Body[0].name.ToUpper() = "LEANNE GRAHAM"'
          httpMethod: Get
          url: https://httpbin.org/
      - name: fleeContainsCheck
        httpRequest:
          skipIf: '$usersQ.Body[0].name.Contains("Graham")'
          httpMethod: Get
          url: https://httpbin.org/
      - name: fleeStartsWithCheck
        httpRequest:
          skipIf: '$usersQ.Body[9].name.StartsWith("Clementina")'
          httpMethod: Get
          url: https://httpbin.org/
      - name: fleeEndsWithCheck
        httpRequest:
          skipIf: '$usersQ.Body[0].email.EndsWith(".biz")'
          httpMethod: Get
          url: https://httpbin.org/
```

## Flee with Json + explicit placeholder quoting

When using Json capture (not QJson), explicitly quote the placeholder before calling C# string methods.

```yaml
name: TestPlan
rounds:
  - name: FeatureRound
    numberOfClients: 1
    arrivalDelay: 1000
    runInParallel: false
    iterations:
      - name: users
        httpRequest:
          capture:
            to: users
            as: Json
            makeGlobal: true
          url: https://jsonplaceholder.typicode.com/users
          method: GET
      - name: fleeQuotedUpperCheck
        httpRequest:
          skipIf: '"$users.Body[0].name".ToUpper() = "LEANNE GRAHAM"'
          httpMethod: Get
          url: https://httpbin.org/
      - name: fleeQuotedContainsCheck
        httpRequest:
          skipIf: '"$users.Body[0].name".Contains("Graham")'
          httpMethod: Get
          url: https://httpbin.org/
```

---

## Notes

- In Flee, compared string values should be quoted, for example "LEANNE GRAHAM".
- Prefer QJson for Flee string methods to avoid manual placeholder quoting.
- For declarative methods examples, use 15.StringMethods.md.
