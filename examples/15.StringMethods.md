# String Methods (Declarative) Complete Examples

This document contains complete runnable plans for declarative string methods only.

---

## tolowercase

```yaml
name: TestPlan
rounds:
  - name: FeatureRound
    numberOfClients: 1
    arrivalDelay: 1000
    runInParallel: false
    iterations:
      - name: users
        httpRequest:
          capture:
            to: users
            as: Json
            makeGlobal: true
          url: https://jsonplaceholder.typicode.com/users
          method: GET
      - name: lowerCaseCheck
        variables:
          - name: lowerName
            value: $tolowercase(source=$users.Body[0].name, variable=lowerName)
        httpRequest:
          skipIf: $contains(source=$lowerName, value=leanne)
          httpMethod: Get
          url: https://httpbin.org/
```

---

## Notes

- Declarative methods are YAML-level functions and do not require C# string-literal quoting.
- contains, startswith, and endswith support ignoreCase=true.
- You can use source= as a named argument or as the first positional argument.
